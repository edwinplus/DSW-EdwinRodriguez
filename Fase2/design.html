Aprender Etiquetas Básicas de HTML

El proposito de este sitio web es aprender cuales son las etiquetas
basicas del HTML identificando su estructura, para alcanzar este objetivo
es de vital importancia conocer el ordenden de la estructura, como funciona, cuando se
debe emplear las diferentes etiquetas que nos ofrece este lenguaje. Por otra parte veremos
el Atributo que son modificaciones que se le pueden realizar a las etiquetas y Valor que es
la especificacion de esta modificacion.

Etiquetas

Etiquetas base: son las primeras etiquetas
que se emplean para crear un archivo HTML. Ejemplo:
  • html: raiz del archivo
  • head: engloba los metadatos
  • title: titulo en la pestaña del navegador
  • Link: enlaza archivos CSS
  • body: elementos visibles en la pagina
Etiquetas de texto: se emplean para dar funciones, jerarquias y estilos diferentes.
  • p: crear parrafos.
  • h1, h2, h3, h4: estrutura titulos y subtitulos, h1 es la mas importante de la jerarquia.
  • b: texto negrillas
  • i: texto cursiva
  • u: texto subrayado
  • a: agregar un link
  • br: crear un salto de linea
Etiquetas multimedia: permite ingresar audio, video e imagenes.
  • img: insertar imagenes
  • video: insertar video
  • iframe: insertar contenido de otras paginas
  • audio: insertar audio.
Etiquetas para estructurar contenido: sirven para ordenar el contenido de la pagina web
  • ul: crea lista sin orden
  • ol: crea lista ordenada
  • li: define articulos de las listas
  • div: agrupa etiquetas
  • nav: crea menu de navegacion
  • header: define la cabecera de la pagina
  • footer: define el pie de pagina
Etiuqeta para crear tablas
  • table: crea una tabla
  • tr: crea las filas
  • td: crea las columnas
  • th: crea el titulo en cada columna
  • cool: especifica propiedades de las columnas
Etiquetas para crear formularios
  • form: crea un formulario
  • input: campo para introducir datos
  • label: el titulo del input
  • button: un boton

Atributos y Valores

Los atributos en HTML son valores adjuntos que ajustan su
su comportamiento de distintas maneras para cumplir criterios
requeridos por el usuario

Algunos atributos empleados en HTML:

  • href =” ”: se usa para enlaces html.
  • src=“ ”: sirve para imágenes.
  • class =” ”: identificador múltiple.
  • border =” ”: sirve para darle borde al contenido.
  • Id =” ”: identifica un elemento único.
  • width =” ”: determina ancho.
  • height =” ”: determina altura.
  • background-color =” ”: color de fondo.
  • style =” ”: sirve para darle un estilo al contenido.
  • align =” ”: alineación de contenido.

Valor de un atributo:

Este representa una cantidad exacta, una caracteristica o una medida
que se ejecuta en este atributo.
Ejemplos:
  • El ancho y la altura se especifican en píxeles de forma predeterminada; width= "100" significa 100 píxeles de ancho.
  • media="screen, 3d-glasses, print and resolution > 90dpi"
  • Un integer es un número entero, como 1024 o -55.
  • 45deg (grados), 5s (segundos) o 10px (píxeles).
  • un conjuntos de cajas, los anchos de las cajas son de 300 píxeles y de 70%, respectivamente.
